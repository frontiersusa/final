Press

A collection of utility styles for Frontiers properties.

Current version: v4.0.0 alpha

# CDN

Get started with Press by including the following code in the head of your HTML file.

<link rel="stylesheet" href="https://cdn.rawgit.com/frontiersusa/press/v4.0/dist/press.min.css"/>

# Installation

Install using NPM (coming soon).

npm install press-css

Or you can download a .zip of the latest version.

Download v4.0 from Github
