Final is a collection of utility styles for Frontiers properties.

Current version: v3.1.0

# CDN

Get started with Final by including the following code in the head of your HTML file.

<link rel="stylesheet" href="https://unpkg.com/final@3.1.0/dist/final.min.css"/>

# Installation

Install/download using NPM.

npm install final-css

Or you can download a .zip of the latest version.

Download v3.0.1
